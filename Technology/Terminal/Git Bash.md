
Git bash is a **bundle** that contains:

1. Bash (the shell)
* The command interpreter
* runs your commands

2. Mintty (the terminal emulator)
* The graphical window where the shell appears
* Handles colors, fonts, resizing, mouse inputs, etc

3. GNU tools for Windows
* ls, grep, sed, ssh, etc
* All compiled for Windows

4. Git for Windows
* Git command-line tools
